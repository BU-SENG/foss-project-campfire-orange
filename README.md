#CAMPUS DELIVERY SYSTEM
## PROJECT OVERVIEW
The system is an open-source web-based platform designed to manage, track, and deliver students' packages within the campus.
## PURPOSE OF THE SYSTEM
The purpose of this project is to streamline campus delivery operations by enabling students and delivery personnel to interact through a centralized online platform. This reduces delays, prevents misplaced items, and enables real-time updates throughout the delivery cycle.
## SCOPE OF THE SYSTEM
The system will support:
### Students:
-Register on the platform
-Request package deliveries 
-Track delivery status
-Receive notifications
### Delivery Personnel:
-View and accept students' delivery requests 
-Update package status in real time
-Record Proof of delivery
## KEY FEATURES:
-Real-time package tracking
-Instant delivery notifications 
-Role-Based Access
-Request History and Delivery logs
-Secure Authentication System
